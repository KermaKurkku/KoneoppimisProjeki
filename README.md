# JEE JEE TENSORFLOW

## Devausohjeita
Kehitys tapahtuu virtuaaliympäristössä

Virtuaaliympäristön luonti
```bash
virtualenv -p python3 .envs/envname
```
envname=projektin nimi

Virtuaaliympäristön käynnistämiseksi 

```bash
cd .envs/envname
source bin/activate
```

Virtuaaliympäristön sulkemiseksi

```bash
deactivate
```
